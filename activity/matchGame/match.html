Rules:


1. There are 16 blocks in this game covered with curtain.


2. In those boxes there are 8 pairs of emojis ,the player has to match the simailar emojis.


3. If the emojis are not similar then emojis curtains up again.


4. If the player guesses all the pairs correctly he wins the game.


5. Try to remember the emoji type to guess correct pair...
